# lineart

The lessons introduce Python and provide material for video animation using points, lines, and rectangles.
The final project shows a video animation of how to plot a house.

[Lesson 1: An introduction to Python](https://github.com/pattichis/lineart/blob/main/LinePlots_Lesson_1.ipynb)
* Variables
* Strings
* Concatenating strings
* Long strings
* For loops
* Using for loops for counting
  
[Lesson 2 and project: Plotting points, lines, and video projects](https://github.com/pattichis/lineart/blob/main/LinePlots_Lesson_2_nogrid.ipynb)
* Object-oriented programming in Python
* Plot points
* Plot lines
* Plot rectangles
* Adding text to your figures
* Draw lines from a given point
* Draw parallel lines
* Draw a house using line art
* Generate frames using line art
* Create a video animation using lien art
